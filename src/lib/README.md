# lib

Leaf utilities and third-party client setup. Modules here should be small, focused,
and have minimal internal dependencies.

- `supabase/` — browser and server (SSR) Supabase clients.
- `stripe/` — Stripe SDK initialization.
- `sanity/` — Sanity client and image/URL helpers.
- `utils.ts` — generic helpers including `cn` (created by shadcn init).
